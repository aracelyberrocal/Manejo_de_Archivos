Informe de desarrollo del proyecto

Introducción

Este proyecto se basa en crear una aplicación de escritorio que se encargará de pedir la descripción de un producto y su respectivo precio. Estos datos se almacenarían al pulsar el botón de "Alta" en un archivo txt que en este caso se llamaría "archivo.txt" , además se mostraría un mensaje de "Guardado " si se ha realizado de forma correcta.

A continuación al pulsar "Visualizar productos" se recogerán los productos con sus precios del archivo guardado y se mostrarían en la interfaz.

He utilizado el IDE de Intellij porque me resulta más comodo al trabajar.

El lenguaje de programación es Java como se pidió desde el principio.

He organizado el codigo en una sola clase y de la siguiente forma:

  • Imports
  • Atributos
  • Lanzadera
  • Métodos
  • Constructor

Metodo readFile()

Este método se encarga de leer las lineas del archivo.txt y posicionarlas en el textArea y lo hará hasta que se acabe la última línea y así cada vez que pulse el botón de "Visualizar Productos"

public void readFile() {
        try {
            Scanner scan = new Scanner(file);
            String fileContent = "";
            while (scan.hasNextLine()) {
                fileContent = fileContent.concat(scan.nextLine()) + "\n";
                txtProductos.setText(fileContent);
            }
            lblResultado.setText("");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }

Metodo writeFile()

Este método escribe la descripción del producto y el precio que intrujo el usuario en los campos de texto, y si todo ha ido correctamente muestra el mensaje de "Guardado" y así cada vez que pulsa el botón de Alta.

private void writeFile() throws IOException {
        try {
            FileWriter myWriter = new FileWriter(file, true);
            myWriter.write(txtDescripcion.getText() + " -> " + txtPrecio.getText() + "€ \n");
            myWriter.close();
            lblResultado.setText("Guardado");
            lblResultado.setForeground(Color.green);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
